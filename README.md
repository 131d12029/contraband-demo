# contraband-demo
「違禁商品通報平台」是一個政府民眾互動型的電商違規檢舉系統，旨在協助社會大眾便捷地通報網路平台上販售的政府公告禁售商品，並提供透明的案件追蹤與統計分析功能。
## 架構設計
